銷貨管理系統使用了兩個核銷欄位,分別為是否核銷與生產核銷
"銷貨核銷"用來檢驗此筆訂單的產品是否已銷貨完畢(0表示未銷貨;1表示已銷貨)
"生產核銷"用來檢驗此筆訂單中的產品是否已生產完畢,分為四個階段(0,1,2,3)

*****此系統主要以無庫存來處理,一般訂貨量=生產量=銷貨量,但仍可修改生產量*****
"0"表示剛產生訂貨單,還未準備生產所需的產品(新增一筆訂單時)
"1"表示產生了生產單,進入生產所需產品的階段(離開訂單時,只要生產核銷=0,即會出現生產單)
"2"表示所需產品已生產完畢,進入銷貨階段(功能表上的"生產核銷"功能),
   這些數量亦同時更新至庫存表的"生產量"中
"3"表示產品已銷出,此筆訂單已核銷(是否核銷=1)(產生銷貨單時),
   同時這些數量也會更新至庫存表的"銷貨量"中,為避免重複計算銷貨量,
   故以"生產核銷=3"來區別準備銷貨(生產核銷=2)與已銷貨(生產核銷=3)的產品
